DeadScout Desktop 0.2.25

Public desktop scope:
- Import local PCAP, PCAPNG, IQ/raw sample, rtl_433 JSON, raw hex, 802.15.4 log, and DeadScout session files.
- Review packet lists, display filters, selected packet details, reports, and spectrum/waterfall snapshots.
- Use RTL/helper and external rtl_tcp paths when configured by the operator outside the public package.

The public Windows package includes app launch files only and does not bundle third-party packet-driver setup or external analyzer links.
