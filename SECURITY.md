# Security Policy

DeadScout is an RF/SDR capture and analysis tool. It does not include telemetry, advertising, account sign-in, or bundled background network services.

## Reporting security issues

Please report security-sensitive issues through the repository owner's preferred GitHub contact path. Do not publish exploit details until the issue has been triaged.

## Distribution status

Release APKs are locally signed for installability. They are not a substitute for independent security review, app-store signing, or real-device hardware validation.
